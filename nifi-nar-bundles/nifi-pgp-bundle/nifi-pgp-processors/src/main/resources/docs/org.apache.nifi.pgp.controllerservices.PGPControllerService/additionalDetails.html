Description

Controller Service that maintains PGP Key Materials.

The controller service supports the following operations via the four related PGP processors:

  • FlowFile content encryption via EncryptPGP
  • Content decryption via DecryptPGP
  • Digital signing of content via SignPGP
  • Digital signature verification via VerifyPGP

Each of these operations requires a specific Controller Service configuration:

  • Encrypt requires either Public Key Text, Public Key File, or a PBE passphrase
  • Decrypt requires either Secret Key Text, Secret Key File, or a PBE passphrase
  • Sign requires Secret Key Text or Secret Key File
  • Verify requires Public Key Text or Public Key File

Public and Secret Keys may be specified directly via the text fields, or as filenames via the file fields. In each of these cases, the specific key the controller uses may be specified by the User ID fields. If no user id is specified, the controller uses the first available key.
